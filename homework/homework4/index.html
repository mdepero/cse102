Creating A Bitmap File with a Hex Editor

The video below demonstrates how to make a bitmap image file using only hex values. This shows how all forms of data are at their root simply binary strings of 1's and 0's formatted in a creative way to make them understandable by computers.

Sources used include the BMP file format Wikipedia article and also the Bless Hex Editor.

The video is also presented in HTML5 rather than with YouTube as HTML5 is easier to implement, let's me customize how the video is presented better, streams faster and in higher quality, and lets me include the video directly on the server.
