One Bit Memory - NOR

This circuit allows a computer to store "memory" by having its output constantly be either 1 or 0 but changable by sending a single pulse to one side or the other. The computer can then read this circuit's value at any time and know what the last digit sent to it was.

One Bit Memory - NAND

This circuit works similar to the XOR memory circuit, except the NAND circuit removes power from either one bit or the other temporarily to change it's bit. So when you press "Turn Bit On" you are changing that input to 0 rather than before you were changing that input to 1.
