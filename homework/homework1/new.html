Built In Date Input

Another cool element in HTML is the Date input. While it isn't very browser compatible (especially with old Internet Explorer) it is a very powerful way to accept dates without even having to write more than one line of code.

See w3school Inputs Tutorial

iFrames


Displaying Matt DePero's CSE102 Home Page

iFrames are a tool that allow you to create a sort of mini browser inside of your website. They can also be used to recieve form data from the web page without having to actually leave the page itself which can be very useful because the user can see the results of sending form data without ever leaving the page.

See w3schools iFrame Tutorial
